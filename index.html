Important Notes:

  • MongoDB Atlas scaling:
    • M40 (8 vCPU, 32GB RAM) for up to 2000 users
    • M50 (16 vCPU, 64GB RAM) for up to 7000 users
    • M60 (32 vCPU, 128GB RAM) for 10000+ users
  • S3 Storage optimization:
    • Lifecycle policies automatically manage tiers
    • 30-day retention in Standard
    • 90-day retention in IA
    • Archive storage in Glacier
  • VPC costs include:
    • NAT Gateway charges
    • Data transfer costs
    • VPN connection
  • Redis Cache scaling:
    • Standard C1 for up to 5000 users
    • Premium tier for 10000+ users
